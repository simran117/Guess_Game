## 📌 About The Project

Guess Game is a simple Python-based console game where the user tries to guess a randomly generated number.

The project demonstrates basic programming concepts such as variables, loops, conditional statements, user input handling, and logical thinking.


## 🚀 Features

✨ Random number generation  
✨ User input handling  
✨ Hint messages (Too high / Too low)  
✨ Attempt counter  
✨ Interactive gameplay
